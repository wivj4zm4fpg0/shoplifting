# csv file abstract
|label|description|
|:---:|:---:|
|id|id|
|video_path|name of video file|
|class|0: no shoplifting, 1: shoplifting|
|start_time|time when action begins|
|end_time|time when action ends|
|x|x coordinate of bounding box|
|y|y coordinate of bounding box|
|width|width of bounding box|
|height|height of bounding box|
|fps|fps of video|

|file name|description|
|:---:|:---:|
|n_frames.sh|フレーム数が表記された「n_frames」ファイルをそれぞれのディレクトリにさくせいする。|
|dl.sh|Youtubeから動画をダウンロードするスクリプト。|
|convert_mp4.sh|動画をmp4に変換するスクリプト。|
